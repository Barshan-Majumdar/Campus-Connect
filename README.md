**Campus Connect** is the 1st-prize-winning project developed for the **Dil Se Design (UI/UX)** competition at **Adamas University's** flagship tech fest, **Signifiya 2026**. 

## ⚡ The "Theft" at Adamas
Representing **Brainware University**, our team, **SBS Dominator 🚩**, entered the competition with one goal: dominance. We successfully secured the **1st Prize**, effectively "stealing" the win on the host university's home ground. 

### 🛡️ The Challenge: Pure Skill, No Shortcuts
The competition was structured to separate prompt-engineers from real designers and developers.

1. **Round 1: The Design Blueprint (Figma)**
   - **Constraint:** Zero AI assistance.
   - **Execution:** We architected a complete "Campus Connect" platform focusing on information architecture, visual hierarchy, and user flow. Every pixel was a manual decision.
2. **Round 2: The Technical Execution (Codebase)**
   - **Constraint:** Real-time development.
   - **Execution:** We translated the static Figma prototype into a functional **codebase**. While most stopped at the design, we delivered a buildable product, clinching the top spot.

---

## 🚀 Key Features
- **Centralized Campus Feed:** Real-time updates for students.
- **Cross-University Networking:** Bridging the gap between students and resources.
- **Resource Repository:** A clean, accessible UI for academic materials.
- **High-Performance Code:** A lightweight, responsive implementation of the Figma vision.

## 🛠️ Tech Stack
- **Design:** Figma (Fundamental UI/UX, 0% AI)
- **Frontend:** HTML5, React, TailwindCSS, Framer Motion, GSAP, Lucid React etc.
- **Architecture:** Component-based design logic.
